Titanic: Machine Learning from Disaster

13th Nov 2015

Titanic: Machine Learning from Disasteris my first Machine Learning project. Implemented most popular algorithms incluing Random Forest, Decision Tree, Linear SVM, Kernal SVM, etc. However, the result is not good, only 0.818. At last, we desioned a vote system, using all the previous results from those models to elect a list of result that most models agree with. Thanks for the vote system, our result rose up to 0.83.

This looks like a small improvement, but according to the fact that we could not get a better result based on one algorithm/model, we actually madea great process. It is easy rising from 0.7 to 0.8, but it is difficult to go higher even if it's a small step.
