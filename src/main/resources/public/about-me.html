Projects

Airbnb New User Bookings

Nov 2015 - Feb 2016, Platform (Kaggle)

This project is a Kaggle competition, and the level is Jobs.

Preprocessing: Combined train and test data set, split those variables into several variables. For instance, use substring() to separate the timestamp into 3 parts, year, month and day. And reduced some noise, like set the age which are larger than 90 and smaller than 14 and missing values as -1.

Training: Used runif() to generate random deviates, and get the traning set and validation set. Used xgboost() to train the model. Predicted values in validation set, and get the importance matrix.

Predicting: Used the top 10 important variables to train the bst model.
